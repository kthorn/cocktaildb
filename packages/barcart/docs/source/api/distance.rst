barcart.distance
================

Core distance computation and recipe analysis module.

This module provides functions for:

- Building ingredient hierarchies and trees
- Computing weighted distances between ingredients
- Building distance and volume matrices
- Computing Earth Mover's Distance (EMD) between recipes
- Finding k-nearest neighbors
- Advanced analytics (expected matches, BLOSUM-like updates)
