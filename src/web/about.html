About Cocktail Database

Welcome to my cocktail database - a hobby project to track cocktails and ingredients.

What You Can Do

Sign up for a free account to:

  • Rate and review cocktail recipes
  • Add personal tags to organize your favorites
  • Track your ingredient inventory
  • Search for cocktails you can make with ingredients on hand

Contributing Recipes & Ingredients

To maintain data quality for analysis purposes (coming soon), only editors can add new recipes and ingredients. If you'd like to become an editor or have recipes to contribute, please contact me at kurt.thorn@gmail.com.

Support & Issues

Found a bug or have a suggestion? Please open an issue on GitHub. As this is a hobby project, I may not be able to fix them immediately, but I appreciate your feedback.

About The Code

The code is open source and available on GitHub. Feel free to fork it and use it for your own purposes.

It was largely vibe coded with Claude, so the code may be a bit messy. The database is a SQLite database, and the API runs via FastAPI on AWS Lambda. It's quite cheap to run.

Cheers and enjoy exploring!
